# TransactionsTransfer

CLI-инструмент для автоматического скачивания банковских выписок из Т-Банк и загрузки транзакций в Google Sheets.

Несколько членов семьи могут использовать этот инструмент — каждый со своей папкой профиля и конфигом — для экспорта своих транзакций в общую Google Sheets таблицу.

## Возможности

- Авторизация через браузер (Chrome/Chromium) с сохранением сессии
- Скачивание выписок операций в формате XLSX за произвольный период
- Автоматическая обработка транзакций:
  - Исключение по шаблонам (например, переводы между своими счетами)
  - Перекатегоризация по настраиваемым правилам
- Дедупликация при загрузке в Google Sheets (на основе хэша строк)
- Пропуск уже скачанных диапазонов

## Установка

```bash
uv sync
playwright install chromium
```

## Настройка

1. **Авторизация в Т-Банк:**
   ```bash
   python -m app login
   ```
   Откроется браузер — войдите в аккаунт. Сессия сохранится в `.tbank_profile/`.

2. **Настройка Google Sheets:**
   - Создайте сервисный аккаунт Google и скачайте JSON-ключ в `.gdrive_creds.json`
   - Укажите URL таблицы в файле `.gdrive_link`

3. **Конфигурация правил обработки** — файл `transactions_config.json`:

   Поля транзакций для сопоставления: `описание`, `категория`, `статус`, и другие из выписки Т-Банк.

   ```json
   {
     "ignore_patterns": [
       {"описание": "между своими счетами"},
       {"категория": "переводы", "описание": "мария з."},
       {"статус": "failed"}
     ],
     "recategorize_patterns": {
       "инвестирование": [
         {"описание": "вывод с брокерского счета"},
         {"описание": "пополнение вклада"}
       ]
     }
   }
   ```

   - `ignore_patterns` — транзакции для исключения (AND для полей в объекте, OR между объектами)
   - `recategorize_patterns` — назначить категорию транзакциям по описанию

## Использование

```bash
# Загрузить транзакции за год
python -m app main --year 2026

# Загрузить транзакции за месяц
python -m app main --year 2026 --month 3
```

## Структура проекта

```
app/
  __main__.py      — CLI интерфейс
  tbank.py         — Экспорт из Т-Банк через Playwright
  transactions.py  — Обработка и хранение XLSX файлов
  google_sheets.py — Загрузка в Google Sheets
```

## Структура файлов

- `transactions/` — скачанные XLSX файлы (по годам и месяцам)
- `.tbank_profile/` — данные сессии браузера
- `.gdrive_creds.json` — ключ сервисного аккаунта Google
- `.gdrive_link` — URL целевой Google Sheets таблицы
- `transactions_config.json` — правила обработки транзакций
